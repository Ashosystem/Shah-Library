The Scatter Approach (NASHR)

This library is organized using the scatter principle - approaching student needs from many different directions with no fixed hierarchy. Start anywhere based on your temperament and interests. The same materials work differently for different readers, and patterns emerge over time through exposure.

  • No hierarchy - start anywhere
  • Multiple entry points based on temperament
  • Non-linear learning approach
  • Patterns emerge through exposure over time
